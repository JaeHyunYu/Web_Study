FlexBox
Box와 값들은 행과 열로 맘대로 배치시켜주는 친구라고함!

1. Container에 들어가는 속성값이 있고, item에 들어가는 속성들이 있음!
2. flex박스에는 중심과 반대축이 있음! 배열되는 애들에 따라 중심축과 반대축이 달라짐
	ex) 애들 배열이 수직으로 된다 하면 수직이 중심축이 되고 수평이 반대축이 되어지고
		반대면 반대로 되는거임!

100%, 100vh!차이!
%는 컨테이너의 부모의 100%를 채우겠다는 의미!
vh는 부모의 상관없이 보이는 전체의 몇퍼센트를 쓰겠따 라는 뜻!

색깔 color tool이라는 곳가면 색깔들 지정해서 보여줌! -> 색깔 코드 복사해서 쓰면된다!

display:flex; 라고 하면 이제 flex박스 쓴다는 뜻임!
flex-direction // 얘는 아이템들 배열해주는 친구! 값들!
	: row; // 왼쪽에서 오른쪽순서로 간다는 뜻
	    : row-reverse; // 오른쪾에서 왼쪽순서로 배치되어짐!
	  : column; // 위에서 아래로 배열!
	:column-reverse; // 아래에서 위로 배열!
flex-wrap : nowrap; // 아이템들이 안겹치도록 하는거! (기본값)
	wrap; // 아이템들이 겹쳐지게 되면 아래로 내려가짐
	wrap-reverse; // 아이템들이 거꾸로 배치되어서 줄 바뀌어짐!
flex-flow : column nowrap; // flex-direction과 flex-wrap 합쳐서 한번에 쓸 수 있는 거!



justify-content // 얘는 아이템들을 수평적에서 배치하는거!
	: flex-start; // row일경우 아이템을 왼쪽에 배치! -> column이면 위에 배치!
	: flex-end; // 오른쪽에 배치! 
	: center; // 가운데 배치!
	: space-around; // 아이템을 감싸는 만큼 떨어뜨려줌 -> 따라서 맨앞과 맨끝은 떨어지는 거가 조금 적음
	: space-evenly; // 모든 아이템 간격이 똑같게!
	: space-between; // 맨 양 끝을 끝으로 붙이고 중간에 있는애들만 띄워지도록 배치!



align-items // 얘는 아이템을 반대축에서 배치하는거!(수직!)
	: baseline // text가 모두 균등하게 보여줌 // baseline에 맞춰서!

align-content // 반대축에 애들을 배치해주는 친구
	: space-between; 양끝애들은 양옆에 이런식으로!
이 값들은 mdn에서 브라우저 체크들 확인하면서 써야됨 새로운애들은 확인해봐야함!!

잘 기억안나는 사이트들은 mdn에서도 확인할 수있고, 여러 사이트들에서도 확인할 수 있음


item 속성값들!
order
	:0 // 기본값으로 그냥 차례대로 가는 거
	:1, 2,3 // 이렇게 순서지정해줘서 아이템들 배치의 순서를 정할 수도 있음!

flex-grow // 아이템 값
	:0 // 기본값
	:1 // 모든 아이템들을 111 이렇게 지정하면 전체 부라우저 비율 1:1:1 이렇게 가져가버리고 그 순서 weight 안드로이드랑 비슷하다고 생각하면됨! -> 엄청 작아지게 되면 결과적으로 비율 다르더라도 똑같이 작아지게되어짐

flex-shrink // 컨테이너가 작아질때 어떻게 작아질지 지정하는애!
	:2 1 1 // 이렇게하면 2인애가 줄어들때 다른애보다 2배더 빠르게 줄어듬

flex-basis // flex-grow와 flex-shrink를 합쳐놓은거라 생각하면됨
	:auto//기본값
	:40% 30% 30% // 이렇게하면 늘어날때도 4:3:3비율유지하고 줄어들때도 유지해서 줄어듬
align-self //아이템별로 아이템들을 정렬할 수 있음
	:center; 하면이 지정한 애만 가운데에 위치해주고 그럴 수 있음
